# 🎓 SkillSphere — Modern Online Learning Platform

### 📝 Project Overview

**SkillSphere** is a modern and fully responsive online learning platform built with Next.js. It allows users to explore skill-based courses, view detailed course information, manage personal profiles, and securely authenticate using email/password or Google login.

The platform focuses on delivering a smooth and engaging user experience with protected routes, animations, responsive design, dynamic rendering, and modern UI components.

This project demonstrates practical implementation of modern frontend and full-stack development concepts including authentication, protected routing, responsive UI design, form handling, search functionality, profile management, and user experience optimization.

# 🚀 Key Features

- **🎓 Skill-Based Courses:** Explore multiple professional courses including Web Development, Design, Marketing, and more.
- **🔥 Popular Courses Section:** Highlighted top-rated courses on the homepage.
- **🔍 Course Search Functionality:** Quickly search courses by title.
- **🔐 Authentication System:** Secure login & registration using BetterAuth.
- **🌐 Google Authentication:** One-click Google sign-in integration.
- **🛡️ Protected Routes:** Course details pages are accessible only to authenticated users.
- **👤 User Profile Management:** View and update profile information dynamically.
- **⚡ Update Profile Feature:** Users can update their name and profile image.
- **🎥 Interactive Hero Section:** Beautiful and modern hero/banner section.
- **📱 Fully Responsive Design:** Optimized for Mobile, Tablet, and Desktop devices.
- **🔔 Toast Notifications:** Elegant feedback system for authentication and profile updates.
- **🎞️ Smooth Animations:** Interactive UI experience using Motion / Swiper.
- **🚫 Custom 404 Page:** User-friendly not-found page implementation.
- **⏳ Loading States:** Smooth loading experience during data fetching.

# 🏗️ How to Run Locally

Follow these steps to run the project on your local machine:

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/mdabdulawal2001/skill-sphere-project-by-toha.git
```

---

## 2️⃣ Navigate to the Project Folder

```bash
cd skill-sphere-project-by-toha
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Setup Environment Variables

Create a `.env.local` file and add the following:

```env
BETTER_AUTH_SECRET=your_secret_key
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
DATABASE_URL=your_database_url
```

---

## 5️⃣ Run the Development Server

```bash
npm run dev
```
